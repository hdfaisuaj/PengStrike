## [0.1.0-alpha] - 2026-06-17

### 新增功能
- **核心功能**：
  - AI 智能对话功能，支持多种 LLM 模型
  - 自动化工具执行功能
  - 报告生成功能
  - 实时 WebSocket 通信

- **配置管理**：
  - 统一配置到 `config.json`
  - 支持通过 Web 界面进行配置
  - IP 自适应配置

- **安装部署**：
  - 一键安装脚本 `install.sh`
  - 一键启动脚本 `Pentest-Web.sh`
  - 自动检测系统环境

- **前端功能**：
  - Vue 3 + Element Plus 前端界面
  - 实时对话界面
  - 工具管理界面
  - 配置管理界面

- **后端功能**：
  - FastAPI 后端 API
  - WebSocket 实时通信
  - 工具注册和执行
  - 报告生成

### 修复问题
- **安全问题**：
  - 修复 CRLF 换行符问题（Windows 编辑 `.sh` 文件导致）
  - 删除假的插件管理功能
  - 添加 LLM Tool Call 清洗白名单
  - 修复前端 WebSocket 连接问题

- **配置问题**：
  - 统一配置到 `config.json`
  - 修复 IP 变化导致连接失败的问题
  - 启动脚本自动检测 IP 并更新配置

- **代码质量**：
  - 修复代码中的安全漏洞
  - 优化错误处理
  - 改进日志系统

### 已知问题
- **Alpha 版本稳定性**：本版本为 Alpha 版本，可能存在稳定性问题
- **工具兼容性**：部分工具可能在不同系统上有兼容性问题
- **LLM 模型支持**：部分 LLM 模型可能支持不完整
- **文档完整性**：部分功能文档可能不完整

### 安全提醒

## 版本说明

- **Alpha**：内部测试版本，功能可能不完整，可能存在较多问题
- **Beta**：公开测试版本，功能基本完整，可能存在一些问题
- **RC**：候选版本，功能完整，主要用于测试
- **Stable**：稳定版本，功能完整，问题较少

## 发布流程

1. **更新版本号**：更新 `config.json` 和 `package.json` 中的版本号
2. **更新 CHANGELOG**：将 `Unreleased` 部分移动到新版本
3. **创建发布标签**：`git tag -a v0.1.0-alpha -m "Release v0.1.0-alpha"`
4. **推送标签**：`git push origin v0.1.0-alpha`
5. **创建 GitHub Release**：在 GitHub 上创建 Release，包含 CHANGELOG 内容
6. **更新文档**：更新 README 中的版本号
